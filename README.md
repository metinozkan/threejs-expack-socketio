# Threejs, Socketio, Express and Webpack Boilerplate

## Installation & Usage

    git clone https://github.com/metinozkan/threejs-expack-socketio.git
    cd threejs-expack-socketio
    npm install

    npm run buildDev

    npm start               // navigate to localhost:8080 for local dev
